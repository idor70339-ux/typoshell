Privacy Policy

Last updated: April 2026
TypoShell does not collect, store, or transmit any personal data. All processing happens entirely within your browser.

What TypoShell does

TypoShell is a Chrome extension that converts mistyped text between English and Hebrew keyboard layouts. It detects wrong-language input and offers automatic corrections as you type.

Data collection

TypoShell does not collect any data. Specifically:

  • No text you type or select is ever recorded or stored.
  • No browsing history or page content is accessed beyond what is necessary to detect selected text.
  • No personal information is collected at any point.

Permissions explained

  • storage — Reserved for future user preferences (e.g. settings). Currently stores nothing.
  • activeTab — Required to interact with the page you are currently viewing.
  • scripting — Required to inject the toolbar and correction UI into web pages.
  • clipboardWrite — Used only to copy converted text when the target element is not editable.

Third-party services

TypoShell uses the free dictionaryapi.dev API to verify whether typed words are valid English. Only the individual word being checked is sent — no surrounding text, no page content, and no identifying information.
